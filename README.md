# 项目概述 python32

本项目是基于Python Flask框架和MySQL数据库开发的图书管理系统，实现了图书管理全流程的功能覆盖，操作便捷且权限管控清晰。

## 技术栈

- Python Flask框架
- MySQL数据库

## 功能模块

### 系统核心功能模块

- 分类管理模块
- 图书管理模块
- 借阅管理模块
- 读者管理模块
- 账号管理模块
- 角色和权限管理模块
- 大模型智能聊天
- 个人中心模块
- 首页展示模块
